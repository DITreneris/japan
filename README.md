# japan

Statinis puslapis, automatiškai publikuojamas per **GitHub Pages** (deploy per **GitHub Actions**).

## Deploy

- Įjunk GitHub Pages repo nustatymuose:
  - **Settings → Pages → Build and deployment → Source: GitHub Actions**
- Push į `main` automatiškai paleis workflow ir atnaujins svetainę.

## Struktūra

- `/` nukreipia į `/en/`
- `en/` – pagrindinis įėjimo puslapis
