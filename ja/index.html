AIに日々の作業の30〜50%を任せる

定型テンプレート8本で、数分で結果が出る演習に沿って進められます。

対象：管理職、専門職、コンサルタント、フリーランス。

無料で始める コミュニティに参加

得られること

30分でAI活用の基本ワークフローを作れます：

  • 週に3〜5時間の節約
  • ミスのリスクを最大6倍低減
  • 標準テンプレート8本
  • その場しのぎの指示ではなく、明確な自動化ロジック

このライブラリの使い方 ステップあたり 約3〜5分

  1. プロンプトを選んでクリックすると、テキストが選択されます
  2. 「プロンプトをコピー」をクリック、または Ctrl+C / Cmd+C
  3. ChatGPT、Claude などのAIツールに貼り付けます
  4. プロンプト内の [COMPANY] は自社またはクライアント企業名に、[MY ROLE] はあなたの役職に置き換えてください。AIの役割（例：「批判的なアナリスト」）は既に指定されているため変更不要です。

使用済み：0 / 8

1
基礎
約3〜5分

AIコンテキスト診断

AIが自社について何を知っていて、どこで誤りやすいかを洗い出す

META: あなたは批判的なビジネスアナリストです。目的：AIが組織について実際に何を知っているのか、どこに確認が必要かを明らかにする。

INPUT: 企業 [COMPANY]。公開情報と一般的な市場知識のみに基づき、あなた（AI）がこの企業をどう理解しているかを説明してください。構成：1) 確実に言えること 2) 推測していること 3) 情報が不足しており誤っている可能性がある点。最後に必ず「以下はユーザーによる確認が必要です。」と明記してください。

OUTPUT: 結果：短く構造化された回答。AIがどこで間違えやすいか、次に進む前に何を確認すべきかが明確になる。

使う前に

こんなとき：組織の分析を始める／AIが会社について何を前提にしているか確認したい／コンテキスト込みでAIを使い始めたい。

事前に置き換え：

  • [COMPANY] → 自社またはクライアント企業名（例：株式会社サンプル）

やること：下のプロンプト本文をコピーして、ChatGPT や Claude などに貼り付けます（アンケートではありません）。

なぜ効くか

AIの「思い込み」や誤判断を減らし、前提のズレを早い段階で可視化できます。

2
分析
約3〜5分

組織ポートレート

会社の輪郭（規模・事業・課題）を短く整理する

META: あなたは経験豊富なビジネスアナリストです。目的：組織のプロフィールとコンテキストを明確にする。

INPUT: 企業 [COMPANY]。以下を分析・要約してください：概算規模（従業員数、売上）；主な事業領域；経営/組織モデル；この業界で典型的な課題。簡潔に、構造化して書くこと。推測している箇所は明示してください。

OUTPUT: 結果：他のプロンプトでコンテキストとして使える「組織の肖像」。

使う前に

こんなとき：会社のプロフィールを短時間で揃えたい／以降のプロンプト用の共通コンテキストを作りたい／顧客・パートナー企業を俯瞰したい。

事前に置き換え：

  • [COMPANY] → 自社またはクライアント企業名（例：株式会社サンプル）

やること：下のプロンプト本文をコピーして、ChatGPT や Claude などに貼り付けます。

つながり

ここで作った「組織の前提」が、以降のプロンプトの精度を底上げします。

3
役割
約3〜5分

組織の中での「私の役割」

目的・責任・影響範囲を、短く実務的に言語化する

META: あなたは組織設計コンサルタントです。目的：ユーザーの組織内での役割（目的・責任・影響）を明確にする。

INPUT: 役割 [MY ROLE]、企業 [COMPANY]。組織分析に基づいて、この役割を説明してください。含める：役割の主要目的；5〜7個の主要責任；意思決定レベル（オペレーション/戦術/戦略）；誰に報告し、日々誰と連携するか。理論は不要、短く具体的に。

OUTPUT: 結果：明確な役割記述。次のステップの参照として使える。

使う前に

こんなとき：自分の役割の定義を揃えたい／新しい会社・新しい職責で起点を作りたい／面談や合意形成のたたき台が欲しい。

事前に置き換え：

  • [COMPANY] → 自社またはクライアント企業名（例：株式会社サンプル）
  • [MY ROLE] → あなたの役職・職能名（例：カスタマーサクセスマネージャー）

やること：下のプロンプト本文をコピーして、ChatGPT や Claude などに貼り付けます。

アウトプット

次のステップ（職務記述・KPI、業務プロセス）にそのまま接続できる「役割の要約」が得られます。

4
人事・文書
約3〜5分

職務記述書 + KPI

実務に直結する職務内容と、測れるKPIをセットで作る

META: あなたは経験豊富なマネージャーです。目的：実務的な職務記述書とKPIを作る。

INPUT: 役割 [MY ROLE]、企業 [COMPANY]。職務記述書を作成してください。含める：主要業務；必要なスキル/コンピテンシー；測定可能なKPIを5〜7個；6か月後の「良い成果」の定義。人事向けの形式より、実際の仕事に焦点を当てる。

OUTPUT: 結果：自己評価やオンボーディングに使える職務記述書。

使う前に

こんなとき：職務記述を整えたい／評価・目標設定のたたき台が欲しい／オンボーディング資料を素早く作りたい。

事前に置き換え：

  • [COMPANY] → 自社またはクライアント企業名（例：株式会社サンプル）
  • [MY ROLE] → あなたの役職・職能名（例：カスタマーサクセスマネージャー）

やること：下のプロンプト本文をコピーして、ChatGPT や Claude などに貼り付けます。

活用の幅

自己点検のメモにも、チーム合意のたたき台にも、そのまま使える実務ドキュメントになります。

5
プロセス
約3〜5分

中核の業務プロセス

時間と集中が向かう「本当に効く仕事」を5つに絞る（80/20）

META: あなたは業務プロセス分析者です。目的：最重要の業務プロセス5つ（パレート80/20）を特定し、時間とエネルギーの使いどころを可視化する。

INPUT: 役割 [MY ROLE]、企業 [COMPANY]。この役割が実際に行っている最重要プロセスを5つ挙げてください。各プロセスについて：目的；主なステップ；関係者；詰まりやすいポイント。説明は20%、具体的な作業の列挙を80%。

OUTPUT: 結果：プロセス一覧。次のステップ（6）で、AIのインパクトが大きい箇所を見つけるために使う。

使う前に

こんなとき：忙しさの正体を整理したい／業務改善の優先順位を付けたい／次の「AI活用」に接続する前提を作りたい。

事前に置き換え：

  • [COMPANY] → 自社またはクライアント企業名（例：株式会社サンプル）
  • [MY ROLE] → あなたの役職・職能名（例：カスタマーサクセスマネージャー）

やること：下のプロンプト本文をコピーして、ChatGPT や Claude などに貼り付けます。

次への布石

プロセスが見えると、AIをどこに入れると最も効くかが一気に判断しやすくなります。

6
AI活用
約3〜5分

AIによる支援と最適化

プロセスに沿って、現場で使えるAIアイデアを具体化する

META: あなたはAI導入コンサルタントです。目的：プロセスを評価し、ChatGPT等のAIツールで改善する具体案を提示する。

INPUT: （すでに記述した）プロセスを評価し、8〜10個の改善アイデアを出してください。各アイデアについて：AIが何をするか；どのプロセス部分が楽になるか；実際の効果（時間/品質/コスト）；実装難易度（簡単/普通/難しい）。非エンジニアにも分かる言葉で。

OUTPUT: 結果：週あたり数時間を節約できる可能性がある、8〜10個の具体アイデア。

使う前に

こんなとき：ステップ5でプロセスが揃ったあと／AIを「どこに」「どう」入れるか具体案が欲しい／週の作業時間を削りたい。

事前に置き換え：

  • （プロンプト本文にプレースホルダが無い場合は不要）直前の出力を、このプロンプトの INPUT に貼り付けます。

やること：下のプロンプト本文をコピーして、ChatGPT や Claude などに貼り付けます。

実務インパクト

「週あたり何時間削れるか」が見える単位で、AIの使いどころを具体化します。

7
ライブラリ
約3〜5分

毎日使うプロンプト集

日常業務の判断・コミュニケーションにそのまま使える短文プロンプトを作る

META: あなたはプロンプト作成者です。目的：この役割と企業のための「日常使い」短文プロンプトを10〜12個作る。

INPUT: 役割 [MY ROLE]、企業 [COMPANY]。計画、問題解決、コミュニケーション、意思決定に役立つプロンプトを作ってください。表形式で提示： [PROMPT] | [WHEN I USE IT] | [PROBLEM IT SOLVES]。

OUTPUT: 結果：個人用のプロンプト集（毎日使える表）。

使う前に

こんなとき：毎回プロンプトを考えるのが面倒／迷ったらすぐ使える「自分専用の型」が欲しい／表形式で整理したい。

事前に置き換え：

  • [COMPANY] → 自社またはクライアント企業名（例：株式会社サンプル）
  • [MY ROLE] → あなたの役職・職能名（例：カスタマーサクセスマネージャー）

やること：下のプロンプト本文をコピーして、ChatGPT や Claude などに貼り付けます。

毎日の時短

「迷う→考える→書く」を短縮し、すぐ実行に移れるプロンプト表が手元に残ります。

8
シミュレーション
約3〜5分

危機シナリオのシミュレーション

プレッシャーと不確実性に備え、先手で行動計画を作る

META: あなたは戦略計画コンサルタントです。目的：重要局面をシミュレーションし、行動計画を用意して、プレッシャーと不確実性に備える。

INPUT: 企業 [COMPANY]、役割 [MY ROLE]。この役割に直接影響する「危機的状況」を想定してください。含める：現実的なシナリオを2つ；最初の14日間の行動計画；AIがどう支援できるか；主要リスクと成功基準。

OUTPUT: 結果：シナリオと14日計画。本番より前にAIで練習して備えられる。

使う前に

こんなとき：危機対応のたたき台が欲しい／プレッシャー下の意思決定を事前に練習したい／AIでシナリオ演習をしたい。

事前に置き換え：

  • [COMPANY] → 自社またはクライアント企業名（例：株式会社サンプル）
  • [MY ROLE] → あなたの役職・職能名（例：カスタマーサクセスマネージャー）

やること：下のプロンプト本文をコピーして、ChatGPT や Claude などに貼り付けます。
